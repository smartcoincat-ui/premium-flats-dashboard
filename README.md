# Premium Flats Dashboard (Docker)

Локальный интерактивный отчёт по премиальным квартирам около Москва-Сити и школы
«Интеллектуал». Готов к запуску в Docker: внутри контейнера крутится Nginx, который
отдаёт статический `index.html` с фильтрами, карточками, фото и ссылками на объявления.

## Как запустить

```bash
cd premium-report-docker
# Собрать образ
sudo docker build -t premium-flats-dashboard .
# Запустить контейнер и пробросить порт 8080 → 80
sudo docker run --rm -p 8080:80 premium-flats-dashboard
```

После старта открой `http://localhost:8080` (или `http://<ip-ноута>:8080`) — увидишь
интерактивную страницу:

- Поиск, фильтр по бюджету и типу (квартира/апартаменты)
- Сортировка по цене/площади/времени до школы
- Фото каждого лота, основные параметры, плюсы/минусы
- Кнопка «Открыть объявление» ведёт на публичную карточку (Neva Towers, Capital Towers и т.д.)

## Обновление данных

1. Обнови файл `index.html` (например, правь массив `listings` внизу).
2. Пересобери образ `docker build -t premium-flats-dashboard .`
3. Перезапусти контейнер.

Фотографии лежат в `assets/` — можно заменить своими (лучше 1600×900 px, формат JPG).

---
Если нужен auto-reload или API подключения, можно быстро обернуть страницу во что-нибудь
вроде Vite/Next, но для локального просмотра и шаринга через Docker этого достаточно.
